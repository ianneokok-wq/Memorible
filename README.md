# ICT 12-D Memories - React SPA with Continuous Audio

## ЁЯО╡ Continuous Background Music Feature

This React Single Page Application (SPA) provides **seamless background music** that flows continuously across all pages without any cuts or interruptions!

### How It Works
- The audio player is mounted at the **App level** (outside the router)
- When you navigate between pages, only the content changes - the audio keeps playing
- A floating music control button appears in the bottom-right corner
- Music starts automatically (if browser allows) or can be started with one click

## ЁЯОо Features

1. **Continuous Audio** - Music plays seamlessly across all pages
2. **Floating Controls** - Music button in bottom-right corner
3. **Volume Control** - Mute/unmute with one click
4. **Protected Routes** - Login required to view memories
5. **GSAP Animations** - Smooth scroll animations on all pages
6. **Firebase Integration** - Online user tracking
7. **Responsive Design** - Works on mobile and desktop
